# nodejs + python爬虫


### 文件
* app.js：启动脚本
* http.py：爬虫脚本
* pyhttp.js：与python脚本通信，进行一个请求，返回网页源代码
* deal.js：处理数据
* result：网页，用来展示数据
* result/result.js：数据
